# Sistema de Gerenciamento de Biblioteca em C

Este projeto consiste no desenvolvimento de um Sistema de Gerenciamento de Biblioteca utilizando a linguagem C, como parte da atividade avaliativa da disciplina.

## ✅ Funcionalidades
- Cadastro de livros
- Cadastro de usuários
- Empréstimos e devoluções
- Pesquisa de livros e usuários
- Relatórios gerenciais
- Persistência de dados em arquivos

## Tecnologias Utilizadas
- Linguagem C
- Bibliotecas padrão (`stdio.h`, `stdlib.h`, `string.h`)

## ▶️ Como Compilar e Executar

```bash
gcc src/main.c -o biblioteca
./biblioteca
